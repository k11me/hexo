composer的一些坑

Posted on 2024-01-15 In Living Disqus:

composer 使用 zip 载入的问题

问题1 no version defined

解决方法为在要载入包的composer.json中添加"version": "1.0"

之后会进入问题2

问题2 Could not parse version constraint : Invalid version string “”

version需要x.x.x模式，三位版本号。

docker 不同网络连接的问题

laravel sail使用的网络为sail，与其他的docker容器并不联通。

这样无法通过内网gitlab获取到私有包。

解决方法是，将gitlab容器添加到sail网络方便访问。

之后是工具就可以发现gitlab加入了sail网络。

其他请求可正常使用。记得为sail添加ssh。

关于 godot 的一些思考

Posted on 2024-01-10 In Living Disqus:

关于 godot 的一些思考

godot uid 与 uuid 绑定

godot为每一个资源都添加一个唯一的uid，

而后端的一些基础物品设置，使用的uuid，两者没用任何关联。

目前想到的办法，是创建一个全局设置文件，将uid与uuid进行绑定。

通过查找uuid ，获取到godot中对应的资源。

godot 玩家物品

当玩家使用，获得物品时，要呈现不同的特性。物品难免有一些通用属性。

这时，创建通用物品类就很重要，其他属性可以再做更多的拓展。
