  2. 引导刷坏的问题

platformio.ini 设置问题

板子是合宙esp32c3简约版，usb直通。

在支持的板子库中并没有。所以要选择esp32-c3-devkitm-1

还需要添加flash mode 为 dio 这点与 Arduino中一致

出现Serial.println()不打印的情况。

添加build_flags标签。-D ARDUINO_USB_MODE = 1与-D ARDUINO_USB_CDC_ON_BOOT=1
