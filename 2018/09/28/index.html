工作详解

  1. 将全局信息显示功能独立
  2. 需要为信息增加分页显示
  3. 搜索结果是否使用通用部分显示
  4. 识别2fa验证信息并相应【全局模组】
  5. 新建信息页面 【需要验证2FA】
  6. 编辑信息页面 【需要验证2FA】
  7. 删除信息页面 【需要验证2FA】

好在2FA与 code message可以使用代码手动添加，如果没先做后端真是难以想象，估计是灾难或者加班地狱吧

反而是语言本地化容易的，可以在休息时完成。

其他 貌似2FA的优先度比较高，但是信息提示也是非常重要

对功能分组编写非常重要！一人并行写是比较容易混乱 笔记是一定要的

好处是发现问题可以及时修正，也要考虑之后的多人协作的效率问题

多是交给组长处理吧
